# Bitcoin Simulator

Dies ist ein interaktives Projekt, das die Grundlagen der Bitcoin-Blockchain durch eine Simulation veranschaulicht. 

## Features

- **Blockchain Simulation:** Erlebe interaktiv, wie durch den Proof-of-Work Konsensmechanismus Blöcke hinzugefügt werden.
- **Lernziele:** Verstehe Core-Komponenten wie Mining, Transaktionen, Difficulty Adjustment, Halving und mehr.
- **Modernes Tech-Stack:** Basierend auf Vite, React und TypeScript.

## Installation & Nutzung

1. Klone das Repository.
2. Installiere die Abhängigkeiten mit `npm install`.
3. Starte die Entwicklungsumgebung mit `npm run dev`.

## Deployment

Die Seite wird automatisch über GitHub Pages (via GitHub Actions) deployed.
